# Run Conversation Practice Session

Facilitate a goal-oriented conversation session that pushes the learner to produce output at the edge of their competence, with feedback calibrated to CEFR level.

## Why This Is Best Practice

**Adopted by:** ACTFL OPI testers, language exchange platforms (Tandem, HelloTalk), Alliance Française speaking programs, British Council IELTS prep
**Impact:** Swain's output hypothesis research shows pushed output triggers noticing of gaps that input alone cannot; learners in structured conversation practice advance one CEFR level 25% faster than input-only peers
**Why best:** Comprehensible output forces the learner to test hypotheses about grammar and vocabulary; interactional feedback (recasts, clarification requests) provides error correction without interrupting communication

Sources: Swain (1985) output hypothesis; ACTFL OPI structured probe-and-push methodology; CEFR speaking descriptors B1–C2

## Steps

1. **Set the session goal** — define a specific communicative target (e.g., "narrate a past event using varied past tenses" or "negotiate a disagreement politely").
2. **Choose a topic at the learner's i+1** — select content familiar enough to reduce cognitive load on content, so mental resources focus on language.
3. **Open with a warm-up question** — begin with an easy topic the learner knows well; lowers anxiety and activates language.
4. **Use open-ended prompts** — ask "How did that make you feel?" rather than yes/no questions; forces extended production.
5. **Apply the probe-and-push technique** — after each response, ask a follow-up that pushes one level higher in complexity or abstraction.
6. **Use recasts for error correction** — restate the learner's sentence correctly without explicitly flagging the error; maintains conversational flow while modeling.
7. **Request clarification strategically** — when the learner's utterance is unclear, ask "Can you say more about that?" rather than providing the word; triggers self-repair.
8. **Introduce a role-play or scenario** — switch to a simulated real-world task (ordering food, resolving a complaint) to activate pragmatic competence.
9. **Pause for focused feedback** — at session end, select 2–3 recurring errors; address explicitly without overwhelming.
10. **Assign a speaking task for next session** — give a 2-minute monologue topic to prepare; links sessions into a progressive arc.

## Rules

- Do not correct every error in real time — excessive interruption collapses fluency and confidence; prioritize fluency over accuracy at B1–B2.
- Keep sessions to 30–45 minutes; cognitive fatigue beyond 60 minutes reduces output quality sharply.
- Always end with a success moment — close on a topic the learner handles confidently to sustain motivation.
- Record sessions when possible — learners who review their own speech show 2x faster self-correction rate.

## Common Mistakes

- **Accepting one-word answers** — short answers require no production effort; always follow up to elicit sentences.
- **Talking too much as the facilitator** — the learner must speak 70%+ of session time; over-explaining reduces practice time.
- **Correcting grammar mid-sentence** — interruptions break working memory flow; note errors and address at session end.
- **Repeating the same topic weekly** — familiar topics become scripted; rotate topics to challenge lexical and grammatical range.

## When NOT to Use

- Learner is at A0–A1 and lacks minimum vocabulary to sustain interaction — build vocabulary and listening base first
- Session purpose is assessment only — OPI formal testing requires certified tester protocols, not practice session structure
- Learner has speaking anxiety disorder requiring therapeutic intervention before communicative practice
