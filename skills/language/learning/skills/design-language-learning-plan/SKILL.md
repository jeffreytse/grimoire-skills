# Design Language Learning Plan

Build a personalized, milestone-driven plan that moves a learner from current level to target proficiency using evidence-based sequencing.

## Why This Is Best Practice

**Adopted by:** ACTFL-certified programs, university language departments, Defense Language Institute, Peace Corps language training
**Impact:** Structured plans reduce time-to-proficiency by 30-40% vs. ad hoc study; CEFR-aligned programs show 85%+ learner goal attainment rates
**Why best:** Explicit goal setting activates monitoring; input-before-output sequencing reduces fossilized errors; spaced milestone reviews sustain motivation

Sources: Krashen (1982) natural order hypothesis; ACTFL OPI proficiency descriptors; CEFR level grid (A1–C2)

## Steps

1. **Assess current level** — administer a placement test or self-assessment grid (CEFR A1–C2) across reading, writing, listening, speaking.
2. **Define target level and deadline** — set a concrete CEFR/ACTFL goal (e.g., B2 speaking in 12 months) to anchor all planning decisions.
3. **Calculate available hours** — multiply weekly study hours by weeks to deadline; benchmark against ACTFL hour estimates per level jump.
4. **Identify skill gaps** — compare current vs. target profiles; rank gaps by impact on communication goals.
5. **Select core resources** — choose one primary textbook or course aligned to target level, plus authentic materials for listening and reading.
6. **Sequence learning phases** — structure plan into: foundation (grammar + core vocabulary), expansion (comprehensible input flood), production (speaking/writing output), refinement (error correction + register).
7. **Set weekly milestones** — define observable checkpoints (e.g., "understand 80% of podcast X by week 6") for each phase.
8. **Build daily schedule** — assign specific activities per day; weight input 70%, output 30% per Krashen's comprehensible input model.
9. **Schedule monthly reviews** — compare actual progress against milestones; adjust resources, pace, or phase length based on data.
10. **Document the plan** — write it down with dates, resources, and metrics so progress is trackable and revisable.

## Rules

- Never skip the placement step — planning from the wrong baseline wastes weeks.
- Keep the target level realistic: ACTFL estimates 600–750 hours for a native English speaker to reach B2 in a Category II language.
- Input must precede and outweigh output in early stages; forcing speaking too early produces anxiety and fossilized errors.
- Revise the plan monthly — a plan that never changes is a plan that stopped working.

## Common Mistakes

- **Setting vague goals** — "learn Spanish" produces no checkpoints and no accountability.
- **Over-relying on a single resource** — one textbook cannot provide the variety of input needed for full proficiency.
- **Front-loading grammar study** — excessive explicit grammar focus delays fluency; grammar should emerge from input exposure.
- **Ignoring speaking until late** — output practice activates noticing; delay past 3 months creates avoidance habits.

## When NOT to Use

- Learner needs a quick phrase sheet for travel, not proficiency development
- Context requires only receptive skills (reading a single document, subtitling one video)
- Learner has reached C2 and seeks cultural refinement, not structured acquisition
